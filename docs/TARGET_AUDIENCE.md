# このSNSが向いているかもしれない人 / 合わないかもしれない人

このページは、Nagi SNS（仮名）の思想に照らして「向いている体験 / 向きにくい体験」を先に言語化するためのものです。  
新規ユーザーが読んでもよい前提で公開します。

---

## このSNSが向いているかもしれない人

- 数や評価をあまり気にせずに発信したい
- 誰かとゆるく、あるいは長期的に関係を育てたい（1対多よりも関係性重視）
- 強い主張や対立、炎上の多い空気に少し疲れている
- 自分のペースで静かに投稿を続けたい
- 通知や反応に振り回されずにSNSを使いたい

---

## 合わないかもしれない人

- 投稿の反応数やフォロワー数の伸びを重視したい
- 多くの人に一度に届くことや拡散されることを重視する
- フォローやブロックなどで人間関係を明確に管理したい
- 刺激やスピード感のあるタイムラインを求めている
- 自分と異なる意見をあまり見たくない

---

## 補足（このSNSが重視していること）

このSNSは、評価や拡散ではなく、ユーザー同士の関係が「1対1のやり取りの積み重ね」によって時間をかけて変化していくことを重視して設計されています。  
関係は固定されず、日々のアクションによって流動的に更新され続けます。
